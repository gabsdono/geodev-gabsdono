# Installation

## Stable release

To install geodev-gabsdono, run this command in your terminal:

```
pip install geodev-gabsdono
```

This is the preferred method to install geodev-gabsdono, as it will always install the most recent stable release.

If you don't have [pip](https://pip.pypa.io) installed, this [Python installation guide](http://docs.python-guide.org/en/latest/starting/installation/) can guide you through the process.

## From sources

To install geodev-gabsdono from sources, run this command in your terminal:

```
pip install git+https://github.com/gabsdono/geodev-gabsdono
```
